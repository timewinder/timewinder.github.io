Game jams!

We occasionally host game jams, which are essentially (fun and friendly) game development competitions! Active and previous game jams will be posted here.

Guidelines:

  • Don't steal others' work
  • Collaboration is allowed (and encouraged)
  • There are no game engine restrictions (eg. Godot, Scratch, P5js, three.js, etc.)


NOTE: If you aren't able to access itch due to it being blocked you can ask us to submit it.

Submissions will close at 12/14 before our meeting!!
